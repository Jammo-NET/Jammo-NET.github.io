Jammo.ParserTools

All of the tools you'd need to write a programming language. For real though, this package removes the need for a tokenizer and lexer implementation in every compiler or interpeter you write. Other features of note include the EnumerableNavigator and EnumerableIterator classes, these make bi-directional enumeration possible with an intiuitive implementation.
